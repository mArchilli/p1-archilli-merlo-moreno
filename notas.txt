## Migraciones: control de mi base de datos. Definen la estructura de la bd utilizando codigo PHP en lugar de usar SQL directamente. Cada migración representa un cambio en la estructura de la base de datos. Por ej: crear tablas, modificar columnas, etc. 

	php artisan migrate 	aplica cambios a la base de datos

## Seeders: se utilizan para poblar la bd con datos iniciales o de prueba. Se ejecutan con el comando. Dentro del Seed podes utilizar Eloquent u otro constructor de consultas de Laravel para crear y guardar registros en la base de datos.
	
	php artisan db:seed	ejecuta el seeder y llena la base de datos
